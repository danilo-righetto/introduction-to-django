# PREPARANDO O AMBIENTE

Criando o ambiente virtual do Python:

```sh
$ python3 -m venv ./venv
```

Para carregar o ambiente virtual use o comando:

```sh
$ source $path/venv/bin/activate
```

Instale o `Django`:
```sh
$ pip install django
```

Para saber se o `Django` foi instalado corretamente use o comando:
```sh
$ pip freeze
```
